# Readme file

Questo è il repository di un progetto di programmazione ad oggetti. Il progetto consiste in un'applicazione Java che simula un sistema di gestione di un ambiente domestico. L'ambiente è composto da stanze, che a loro volta contengono dispositivi e sensori. I dispositivi possono essere accesi o spenti, mentre i sensori possono essere attivi o inattivi. L'applicazione permette di accendere e spegnere i dispositivi.
